PyScript

  • PyScript는 프레임워크로 HTML에서 python을 이용하여 다이나믹 어플리케이션을 빌드할 수 있게 하는 도구이다.

이벤트를 이용한 기본 예제 코드

  • index.html에서 숫자를 선택하고 버튼을 클릭하면, 이벤트가 발생하여 main.py의 메소드가 실행된다.
  • main.py에서는 동작을 수행하고, 결과는 index.html 태그에 결과 메시지를 저장한다.
